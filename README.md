# Complaint Management System (CMS)

A full-stack Complaint Management System that allows users to register complaints and enables admins to manage, track, and resolve them efficiently.
The Complaint Management System is designed to simplify the process of lodging and handling complaints.
It provides a structured workflow for users and administrators with proper authentication and role-based access.

# Tech Stack
## Backend

Java

Spring Boot

Spring Data JPA

Spring Security

MySQL

Hibernate

Maven

# Features
## User

User registration & login

Submit complaints

View complaint status

Update profile

## Admin

View all complaints

Update complaint status

Manage users

Dashboard overview

# Security

JWT-based authentication

Role-based authorization (USER / ADMIN)

Secure REST APIs

# Database

MySQL

JPA entities with Hibernate ORM

Auto schema generation
